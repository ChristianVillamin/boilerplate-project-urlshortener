URL Shortener Microservice

A freeCodeCamp API Project by @villamin_c. Link to GitHub.

Usage

The API endpoint is POST /api/shorturl/new.
Responds with the original url and the short url number code.


Use the base url (https://cv-url.glitch.me) and add /<number code>.
E.g. cv-url.glitch.me/0

Short URL Creation POST Form

Example

Long URL

https://www.amazon.com/Clean-Code-Handbook-Software-Craftsmanship/dp/0132350882/ref=sr_1_1?keywords=clean+code&qid=1564044404&s=gateway&sr=8-1

Short URL

https://cv-url.glitch.me/1

User Stories

  1. I can POST a URL to [project_url]/api/shorturl/new and I will receive a shortened URL in the JSON response. Example : {"original_url":"www.google.com","short_url":1}

  2. If I pass an invalid URL that doesn't follow the http(s)://www.example.com(/more/routes) format, the JSON response will contain an error like {"error":"invalid URL"}

  3. When I visit the shortened URL, it will redirect me to my original link.
